# 上空云服务 Android SDK

## 安装
在Android Studio中在项目 `build.gradle` 文件 `dependencies` 中增加如下代码:   

```
    compile 'com.android.xxx.xxx'
    compile 'com.android.support:appcompat-v7:22.2.0'
    compile 'com.loopj.android:android-async-http:1.4.5'
```



## 使用说明
